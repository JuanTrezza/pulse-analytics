## Sobre el proyecto

PULSE Analytics es un dashboard de analytics para e-commerce orientado a equipos que necesitan observar el rendimiento comercial, el comportamiento del cliente y la evolución del funnel desde una interfaz clara y accionable.

## Tecnologías utilizadas

- React
- TypeScript
- Tailwind CSS
- Recharts
- Vite
- GitHub Pages

## Características

- KPIs en tiempo real
- Gráficos interactivos
- Embudo de conversión
- Segmentación de clientes
- Reportes operativos

## Modelo de negocio

B2B SaaS

3. Run the app:
   `npm run dev`
